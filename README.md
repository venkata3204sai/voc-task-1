A clean, responsive **personal portfolio website** built with HTML and CSS.  
The site features sections for About Me, Skills, Projects, and Contact, with smooth scrolling navigation and modern UI styling.

## Features

### Fully responsive design  
Built with flexbox and CSS grid to adapt across desktop, tablet, and mobile.

### Sections included
- **Header + Navigation**
- **About Me** (profile picture + intro)
- **Skills & Interests**
- **Projects section** (template cards for future links)
- **Contact form**
- **Footer with copyright**

### Smooth UX enhancements
- Hover animations on cards and buttons  
- Clean UI styling with custom CSS variables (color theme)  
- Semantic HTML structure for readability and accessibility

## Running the Project

Just open the HTML file:
```bash
git clone https://github.com/venkata3204sai/voc-task-1
cd voc-task-1
open index.html
```
OR run a static server:
```bash
python -m http.server 8000
# visit http://localhost:8000
```

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Custom properties)
- No frameworks, no JavaScript required
